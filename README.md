# 🎓 Fatec-DSM - Portfólio de Estudos

Bem-vindo ao meu repositório centralizado de matérias do curso de **Desenvolvimento de Software Multiplataforma (DSM)** na **Fatec Jacareí**. 

Este espaço foi criado para organizar todos os projetos, exercícios e aprendizados adquiridos ao longo da graduação, servindo como um histórico da minha evolução como desenvolvedor.

---

## 🏛️ Sobre o Curso
O curso de DSM foca no ciclo completo de desenvolvimento de software, abrangendo desde a concepção e arquitetura até a implantação em diferentes plataformas (Web, Mobile e Desktop), utilizando metodologias ágeis e tecnologias de ponta.
